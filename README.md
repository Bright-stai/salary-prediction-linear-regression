🚀 Salary Prediction using Linear Regression (One Variable)

This project demonstrates how to build a simple Linear Regression model from scratch using Python to predict salary based on years of experience.

📊 Problem Statement

Given a dataset of employee experience and salary, the goal is to predict salary using a single feature:

Experience → Salary

🧠 What I Learned
Understanding training examples

Implementing the model:

y = wx + b
Writing a cost function
Implementing gradient descent step-by-step
Importance of feature scaling
Visualizing model predictions
⚙️ Technologies Used
Python
NumPy
Matplotlib
📈 Model Visualization

The model learns by minimizing cost over iterations and fitting a line through the data.

▶️ How to Run
pip install -r requirements.txt
python salary_prediction.py
